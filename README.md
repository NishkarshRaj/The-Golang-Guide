## The Golang Guide

![Cover Image](docs/cover.gif)

### Running Golang on *Nix

#### Installation

```
$ apt install -y golang-go
```

#### Check version

```
$ go version
```

#### Standardize File Format

```
$ go fmt [filename]
```

#### Create a Platform Independent Build

```
$ go build [filename]
$ ./[filename]
```

#### Execute a golang program

```
$ go run [filename]
```
